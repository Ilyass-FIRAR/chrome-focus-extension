# Focus Mode Chrome Extension

A simple Chrome extension built as my **first hands-on project** to learn how browser extensions work.
This project follows a **learning-by-doing approach**, where I build the extension step by step while documenting the development process on GitHub.

## Purpose

The goal of this project is to understand the **core architecture of Chrome extensions**, including Manifest V3, popup interfaces, content scripts, and Chrome APIs.

## Features

* Toggle **Focus Mode** from the extension popup
* Block distracting websites (e.g., YouTube)
* Simple and lightweight interface

## Technologies Used

* JavaScript
* HTML / CSS
* Chrome Extensions API
* Manifest V3

## Installation

1. Clone the repository
2. Open Chrome and go to:

```
chrome://extensions/
```

3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder

This repository documents my **learning journey in Chrome extension development**, where each commit represents a step forward in understanding how extensions are built and structured.
